# Soltine - 자연의 아름다움

[Soltine] 브랜드 공식 웹사이트

## 🌐 사이트 주소

GitHub Pages로 배포된 사이트: `https://your-username.github.io/your-repo-name/`

## 📁 프로젝트 구조

```
├── index.html              # 홈 페이지
├── products.html           # 제품 페이지
├── ingredients.html        # 성분용어집 페이지
├── about.html              # 브랜드 소개 페이지
├── contact.html            # 연락처 페이지
├── product-detail.html     # 제품 상세 페이지
├── styles.css              # 메인 스타일시트
├── script.js               # 메인 JavaScript
├── chatbot.js              # 챗봇 JavaScript
└── images/                 # 이미지 및 비디오 파일
    ├── model*.jpg          # 모델 이미지
    ├── product*.png        # 제품 이미지
    ├── product*.mp4        # 제품 비디오
    ├── ingre_*.png         # 성분 이미지
    ├── ingre_*.mp4         # 성분 비디오
    └── ...
```

## 🚀 배포 방법

자세한 배포 방법은 `GITHUB_PAGES_DEPLOY.md` 파일을 참고하세요.
